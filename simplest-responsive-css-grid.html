最简单的响应式表格布局

官方网址

利用的原理就是table-layout：fixed

			@media (min-width: 30em) {
			    .row { width: 100%; display: table; table-layout: fixed; }
			    .col { display: table-cell; }
			}

css代码也奉上

			.col:nth-of-type(2n) {
			    background-color: #95cbcb;
			}
			.col {
			    height: 4em;
			    vertical-align: middle;
			    line-height: 4em;
			    background-color: teal;
			    text-align: center;
			}

30em就是 30*16=480px 也就是当小于480px的时候 就恢复了div本身的状态
